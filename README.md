# Loan Default Prediction System

This project implements a machine learning pipeline to predict the likelihood of loan default using structured financial data. The system applies classification algorithms to identify high-risk applicants and support data-driven decision-making.

## Key Features
- End-to-end ML pipeline: data preprocessing, feature engineering, model training, and evaluation  
- Implemented models: Logistic Regression and Random Forest  
- Handled class imbalance to improve prediction performance  
- Evaluated using ROC-AUC score, confusion matrix, and classification metrics  

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  

## Objective
To build a reliable classification system that helps financial institutions assess credit risk and minimize potential losses.
